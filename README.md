# MMstack Website

Site vitrine de la startup MMstack — Ambam, Cameroun.

## Stack
- React + Vite
- Tailwind CSS v3 (dark mode class)
- Framer Motion (animations)
- Formspree (formulaire contact)
- Lucide React (icônes)

## Démarrage local
```bash
npm install
npm run dev
```

## Déploiement Vercel
1. Push sur GitHub
2. Importer le repo sur vercel.com
3. Framework preset : Vite
4. Deploy — c'est tout !
